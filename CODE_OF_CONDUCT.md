# Code of Conduct

## Our Pledge

We are committed to making participation in WiseMindOS a respectful, inclusive, and collaborative experience for everyone, regardless of background, identity, experience level, nationality, or skill set.

Our goal is to foster an open-source environment where contributors can learn, collaborate, and grow together while building meaningful software.

---

# Expected Behavior

Contributors are expected to:

- Use respectful and professional communication
- Be welcoming to contributors of all experience levels
- Provide constructive feedback during discussions and code reviews
- Respect differing opinions and technical perspectives
- Focus on collaboration and project improvement
- Maintain patience and professionalism during discussions
- Ask questions whenever requirements or expectations are unclear
- Support a positive and growth-oriented community environment

---

# Unacceptable Behavior

The following behaviors are unacceptable within the WiseMindOS community:

- Harassment, discrimination, or hate speech
- Personal attacks, insults, or intimidation
- Toxic, hostile, or disruptive behavior
- Publishing private or confidential information without permission
- Spam, trolling, or intentionally misleading contributions
- Repeatedly ignoring project guidelines or maintainer instructions
- Sharing credentials, secrets, API keys, or sensitive information publicly
- Any behavior that could make contributors feel unsafe, unwelcome, or uncomfortable

---

# Contribution Standards

All contributors are encouraged to:

- Follow repository contribution guidelines
- Keep discussions constructive and project-focused
- Submit meaningful and well-tested contributions
- Respect maintainers, reviewers, and fellow contributors
- Maintain professionalism in issues, pull requests, and discussions

---

# Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Maintainers may take appropriate action against unacceptable behavior, including:

- Warning contributors
- Removing comments or discussions
- Closing issues or pull requests
- Rejecting contributions
- Temporarily or permanently restricting participation

Enforcement decisions will be made with fairness and in the best interest of maintaining a healthy community environment.

---

# Reporting Issues

If you experience or witness unacceptable behavior, please report it respectfully through the maintainer contact channels listed in the repository README or project discussions.

All reports will be reviewed carefully and handled appropriately.

---

# Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues
- Pull requests
- Discussions
- Project documentation
- Community interactions
- Any other official WiseMindOS communication channels

---

# Acknowledgement

By participating in WiseMindOS, contributors agree to follow this Code of Conduct and help maintain a positive and respectful open-source environment.
